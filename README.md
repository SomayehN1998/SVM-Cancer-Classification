# پروژه‌های یادگیری ماشین دانشجویی

مجموعه‌ای از پروژه‌های عملی یادگیری ماشین که دقیقاً طبق صورت سوالات تمرینات دوره پیاده‌سازی شده‌اند. این ریپازیتوری به‌عنوان پورتفولیو دانشجویی، شامل تکنیک‌های پیش‌پردازش داده، طبقه‌بندی، خوشه‌بندی، کاهش بعد، پردازش تصویر و انتخاب ویژگی است.

هدف: ارائه کد تمیز، مستند و حرفه‌ای بدون نمایش نتایج عددی یا خروجی‌ها — فقط پیاده‌سازی صحیح و حرفه‌ای هر سوال.

## فهرست پروژه‌ها

### ۱. پیش‌بینی باران فردا با الگوریتم KNN
- پیاده‌سازی کامل مدل K-Nearest Neighbors
- پیش‌پردازش داده (مدیریت مقادیر گم‌شده، کدگذاری OneHotEncoder)
- آموزش مدل، ارزیابی با Accuracy و F1 Score
- رسم Confusion Matrix، ROC Curve و توزیع کلاس‌ها

### ۲. مقایسه رگرسیون لجستیکی و Naive Bayes
- بارگذاری و پیش‌پردازش دیتاست Veros
- آموزش و ارزیابی هر دو مدل
- رسم ماتریس درهم‌آمیزی برای هر مدل
- مقایسه نهایی و اعلام مدل برتر با دلیل

### ۳. فشرده‌سازی تصویر با الگوریتم K-Means
- بارگذاری تصویر و تبدیل به آرایه دوبعدی
- اعمال K-Means برای کاهش تعداد رنگ‌ها (Quantization)
- بازسازی تصویر فشرده‌شده
- آزمایش با مقادیر مختلف K و مقایسه حجم فایل

### ۴. کاهش بعد با PCA و LDA روی دیتاست دیابت
- خواندن دیتاست با pandas
- تحلیل آماری با numpy
- اعمال PCA (غیرنظارتی) و LDA (نظارتی)
- نمایش بصری داده‌ها در فضای کاهش‌یافته

### ۵. استخراج کانال‌های رنگی RGB تصویر
- بارگذاری تصویر با Pillow
- جداسازی کامل کانال‌های قرمز، سبز و آبی
- ذخیره هر کانال به‌صورت فایل جداگانه

### ۶. تولید آرایه‌های تصادفی و DataFrame
- تولید سه آرایه تصادفی x، y، z با np.random.randint
- ساخت DataFrame با pandas
- نمایش جدول داده‌ها

### ۷. انتخاب ویژگی Forward Selection با SVM روی دیتاست سرطان سینه
- بارگذاری و پیش‌پردازش دیتاست cancer.csv
- پیاده‌سازی Sequential Feature Selector (Forward)
- آموزش SVM با و بدون انتخاب ویژگی
- مقایسه عملکرد و تحلیل تأثیر انتخاب ویژگی
