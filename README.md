# پروژه‌های یادگیری ماشین بدون نظارت

این مخزن شامل راه‌حل‌هایی برای چندین مسأله یادگیری ماشین است که بر تکنیک‌های یادگیری بدون نظارت تمرکز دارند. وظایف شامل پیش‌پردازش داده‌ها، کاهش ابعاد، انتخاب ویژگی‌ها و خوشه‌بندی است. در هر یک از این وظایف، از مجموعه داده‌های مختلف و روش‌های متفاوتی استفاده شده است تا قدرت یادگیری بدون نظارت را نشان دهد.

## پروژه‌ها

### 1. **پیش‌پردازش داده‌ها و اعمال تکنیک‌های کاهش ابعاد (PCA و LDA)**

در این مسأله، از یک مجموعه داده (`diabetes.csv`) استفاده می‌کنیم. پیش‌پردازش داده‌ها، مدیریت مقادیر گم‌شده و سپس کاهش ابعاد با استفاده از تکنیک‌های LDA (تحلیل تمایز خطی) و PCA (تحلیل مولفه‌های اصلی) انجام می‌شود. این داده‌ها قبل و بعد از اعمال تکنیک‌های کاهش ابعاد، تجسم می‌شوند.

### 2. **ایجاد داده‌های مصنوعی و پردازش آن‌ها برای تحلیل‌های بعدی**

در این بخش، یک مجموعه داده مصنوعی ایجاد می‌کنیم. هدف ایجاد داده‌های تصادفی با استفاده از `np.random.randint` و سپس انجام پیش‌پردازش داده‌ها است. این شامل تولید دو ویژگی، محاسبه فواصل و انجام تبدیل داده‌ها برای تحلیل‌های بیشتر است.

### 3. **انتخاب ویژگی‌ها برای مجموعه داده سرطان با استفاده از روش‌های مختلف**

در این مسأله، از مجموعه داده (`cancer.csv`) برای اعمال تکنیک‌های انتخاب ویژگی استفاده می‌کنیم. هدف اصلی انتخاب ویژگی‌های مهم با استفاده از روش انتخاب پیشرفته است و عملکرد این روش را با مقایسه نتایج قبل و بعد از انتخاب ویژگی‌ها ارزیابی می‌کنیم.

### 4. **انتخاب ویژگی‌ها با استفاده از `sklearn` و ارزیابی مدل‌ها**

در این بخش، از انتخاب ویژگی برای یک مجموعه داده استفاده کرده و سپس مدل‌های ارزیابی را برای سنجش عملکرد مدل به کار می‌گیریم. انتخاب ویژگی با استفاده از ماژول `feature_selection` در `sklearn` انجام می‌شود و ارزیابی مدل‌ها نیز با و بدون انتخاب ویژگی مقایسه می‌شود.
